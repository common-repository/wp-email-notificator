== Description ==

This plugin notifies by email in the WordPress.com way to all your subscriptors. It notifies in its first release:

* Posts
* Edits in posts
* Pages
* Edits in pages
* Comments
* Edits in comments

The comments are shown threaded.

New options to come. See roadmap.

== Installation ==

1. Upload `wp-notificator` folder to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
